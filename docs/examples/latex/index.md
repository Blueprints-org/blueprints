# LaTeX and Language Examples

This folder contains examples demonstrating how to change the language settings and how LaTeX content can be converted to Word format using the Blueprints library. These examples are useful for users who need to work with multilingual documents or require conversion of mathematical expressions and formulas from LaTeX to Word for reporting and documentation purposes.

## Available Examples

- **[Change Language](change_language.ipynb)** – Learn how to switch between different languages in your documents and outputs.
- **[Convert LaTeX to Word](latex_to_word.ipynb)** – See how mathematical expressions written in LaTeX can be automatically converted to Word-compatible equations.
